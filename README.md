# 住所を入れるだけの移動しやすさ診断（japan-mobility-ease-diagnosis）

**公共交通オープンデータチャレンジ2026 応募作品（開発中）**

住所を入力するだけで、その場所の「移動しやすさ」を診断カルテ形式で表示するウェブアプリ。

- 最寄りバス停・鉄道駅までの**道路距離ベースの徒歩時間**
- そのバス停の**平日1日あたり運行本数**（バス停はあっても1日2本、を可視化）
- **デマンド交通（乗合タクシー等）のカバー有無**（GTFS-Flex）
- 病院・スーパーへの**公共交通+徒歩での到達可能性**

ベースは [japan-transit-desert-analysis-125](https://github.com/shiwaku/japan-transit-desert-analysis-125)（全国125mメッシュ×道路距離Dijkstraによる公共交通空白地域分析、空白人口1,772万人を推計）。本リポジトリはこれを「空白マップ」から「移動しやすさ診断」アプリへ発展させる。

## なぜ作るか

- 全国の食料品アクセス困難人口は904万人（65歳以上の4人に1人）、うち63%が75歳以上。免許返納は年43.5万件で、毎年40万人超が新たに「クルマなし」の生活へ移行している
- それにもかかわらず、**自治体の約半数は移動困難の実態を把握していない**（総務省行政評価局調査）。「困っている人が可視化されていない」こと自体が課題
- 「住所を入れるだけ」の診断は、この可視化ギャップを当事者・家族・自治体の全員に対して埋める

## データソース

| データ | 提供元 | ライセンス |
|---|---|---|
| バスGTFS-JP | [GTFSデータリポジトリ](https://gtfs-data.jp/) / 公共交通オープンデータセンター | 各データのオープンライセンス |
| GTFS-Flex（デマンド交通） | [公共交通オープンデータセンター](https://www.odpt.org/) | CC BY 4.0 |
| 鉄道駅（S12）・道路（N13）・医療機関（P04） | 国土数値情報 | 国土数値情報利用約款 |
| 125m/250mメッシュ人口（年齢別） | e-Stat 令和2年国勢調査 | 政府標準利用規約 |
| 食料品店POI | OpenStreetMap | ODbL |

> **食料品店POIの選定メモ**: Yahoo!ローカルサーチAPI（YOLP）は2022年12月施行の規約第6条で取得データの保存・キャッシュが禁止されており、タイル化・再配布する本用途には使用不可。OSMは日本のスーパー約94%・コンビニ約84%をカバーし（OSM Wiki / チェーン店協会統計比）、ODbLで再配布可能なため主データとする。カバレッジ検証にはOverture Places（CDLA-Permissive-2.0）と経済センサス小売業メッシュ（2025年10月提供開始）を用いる。
