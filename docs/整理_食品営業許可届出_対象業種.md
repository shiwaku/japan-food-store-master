# 整理：食品営業許可・届出の対象業種

> 対象: 改正食品衛生法（令和3年6月1日施行）の営業規制。データは大橋本家 `japan-facilities-address` /
> 厚労省 i2fas の「営業の種類」列に対応。
> 作成日: 2026-07-18
> 一次情報: 厚労省「営業規制（営業許可、営業届出）に関する情報」／徳島保健所（許可32業種）／江東区（届出・届出不要）
> 関連: `調査_大橋本家japan-facilities-address_取得構造と網羅性.md` / `整理_市区町村別_収集経路とジオコーディング.md`

## 0. 制度の3層構造（令和3年6月改正）

改正で、旧34許可業種を再編し **許可32業種**へ。加えて **営業届出制度**を新設。事業者は次のいずれか。

| 層 | 内容 | 手続き |
|---|---|---|
| ① **要許可業種（32）** | 食中毒リスク・規格基準等が高い業種 | 保健所の**営業許可**（更新あり） |
| ② **要届出業種** | 許可対象でも届出不要でもない全事業者 | 保健所への**営業届出**（更新なし・営業開始届） |
| ③ **届出不要業種** | リスクが低い／他法令で管理される等 | 手続き不要 |

> データ（i2fas / 各自治体OD）の「営業の種類」列には**許可業種と届出業種が混在**する。
> **丸数字が2系統ある点に注意**: 「①飲食店営業…㉜添加物製造業」は**許可番号**、
> 「①魚介類販売業(包装済)…㉙その他」は**届出番号**で、番号が重複する（例: ㉔は許可=麺類製造業／届出=その他の食料品製造・加工業）。

## 3. 届出不要業種

以下はリスクが低い／他法令等で管理されるため、許可も届出も不要。

- 食品又は添加物の**輸入業**
- 食品又は添加物の**運搬業・貯蔵業**（冷凍・冷蔵を除く）
- **常温で長期保存可能**な包装食品のみの販売業
- **器具・容器包装の製造業**（合成樹脂以外のもの）
- 器具・容器包装の**輸入・販売業**
- 農家・漁家の**採取・出荷**、学校・病院等以外の**特定給食でない給食** 等

## 4. データ利用上の注意

1. **2系統の丸数字が同一列に混在**（許可①〜㉜／届出①〜㉙）。集計時は業種名（文字列）で寄せるのが安全。番号だけで寄せると許可と届出が衝突する。
2. i2fas の `_all` ファイルは**許可・届出・廃業を含む全件**。`廃業年月日`（列21）が入る行は廃業済み。
   現役のみ見るならここで除外する。
3. `営業の種類`（列6）とは別に **`業態`（列7）** の副分類列がある（より細かい実態区分。集計軸として併用可）。
4. スナップショット時点はソースにより異なる（京都市 R3.3末 等）。改正前の旧業種名が残るデータもあり得る。
5. **本メモの件数は北海道域3ファイルのサンプル**（57,684件）。全国分布は build 完了後に `facilities-all.csv` の
   `business_type` で全国集計して置き換える余地あり。
