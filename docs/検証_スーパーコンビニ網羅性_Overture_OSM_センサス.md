# 検証：スーパー・コンビニの網羅性（Overture / OSM × 経済センサス）

> 対象: `data/overture_food_deduped_jp.parquet`, `data/osm_food_stores_japan.tsv`
> 正解データ: 令和3年 経済センサス‐活動調査（e-Stat 表 `0004003257` 細分類×都道府県）＋ JFA（コンビニ実店舗）
> 手法: 都道府県境界（dataofjapan/land japan.geojson, 47）で `ST_Contains` 空間結合 → 都道府県別集計
> 作成日: 2026-07-18 ／ 関連: `検証_飲食店網羅性_経済センサス突合.md` / `検証_食品店データ_OSM_vs_Overture.md`

## 結論

**買い物難民診断の本命であるスーパー・コンビニは、POI（Overture/OSM）が許可データより圧倒的に網羅的。**
特に **Overture のコンビニは実数（JFA約56,000）にほぼ一致**し、悉皆ソースとして妥当。
（対して許可データ i2fas のコンビニは 27,615＝実数の約半分。`検証_食品営業許可届出データ_網羅性と検索API適性.md`）

## 全国合計（令和3年基準）

### コンビニ
| ソース | 件数 | 実数比 |
|---|---:|---:|
| **業界実数：商業動態統計 都道府県別店舗数(2021)** | **56,352** | 100%（＝JFA約56,000と一致） |
| **Overture** convenience_store | **54,793** | **97%** ◎ |
| OSM shop=convenience | 48,602 | 86% ○ |
| 経済センサス 5891 | 22,714 | 41% ✗（過少計上・下記） |

> **★コンビニの都道府県別“業界実数”は経産省「商業動態統計」にある**（e-Stat 表 `0004010382` 等、cat02=店舗数、
> 47都道府県、月次・全数ベース）。全国56,352はJFA約56,000と一致＝これが再現可能な正解データ。
> **センサス5891が過少なのは既知の癖**（FC店が本部分類や581側に計上され細分類5891に載らない）。→ コンビニはセンサス不可、商業動態/JFAが正。
>
> **都道府県別カバー率（vs 商業動態実数）**: 全国 **Overture 97% / OSM 86%**。Overtureは県ごとも0.87〜1.16と均質＝
> **都道府県レベルでも実数の約97%を捕捉**（>1.0の県は閉店・重複の可能性）。詳細 `検証_コンビニ網羅性_商業動態実数_都道府県別.csv`。

### スーパー
| ソース | 件数 | 対象/備考 |
|---|---:|---|
| **業界実数（SM3団体）** | **23,078** | 全国スーパーマーケット協会「数字でみるスーパーマーケット2023」（白書2024）。大型1,796/中型15,973/小型5,309、総販売額25.5兆円 |
| 経済センサス 581 各種食料品小売業 | 22,378 | **業界実数の97%＝ほぼ一致 ◎** |
| OSM shop=supermarket | 20,319 | 業界実数の88% |
| Overture supermarket | 18,521 | 業界実数の80% |
| Overture supermarket+grocery_store | 39,031 | grocery_store 20,608 込み（小規模店も捕捉） |
| （参考）経済センサス 561 百貨店・総合スーパー | 1,023 | GMS/百貨店は別カテゴリ |

> **スーパーはセンサス581を正としてよい（実数で裏取り済み）**：センサス581(22,378)が業界実数(23,078)の97%。
> コンビニ（センサス5891が実数の41%）と対照的。
> カテゴリ対応の注意：Overture`grocery_store`は小規模食料品店を含みセンサス581を上回る。OSM supermarketは業界実数の88%でよく一致。
> 買い物アクセス目的ではgrocery_storeも「食料調達点」として有効なので、Overtureの広めの捕捉はむしろ利点。
> **将来**：2024年4月JSIC改定で細分類「5811 食料品スーパーマーケット」新設（従来581は個人商店混在）、2026年経済センサスで食品スーパー業界規模が正確把握可能に。

## 都道府県別の傾向

- **コンビニ OSM/Overture 比 = 全国0.89**。ただし**地方でOSMが薄い**（岐阜0.58・長崎0.59・大分0.60・佐賀・熊本・新潟・富山・山口 ≈0.6）。
  → OSMは都市部は充実だが地方はOvertureの約6割。**Overtureの方が全国均質に網羅**。
- **スーパー**：OSM supermarket ≈ センサス561+581（例 神奈川 OSM1,325/ｾﾝｻｽ1,324、兵庫 978/987、愛知 767/971）。
  Overtureは grocery_store 込みで一貫して多い（東京 Ov5,016 vs OSM2,620 vs ｾﾝｻｽ2,179）。

（全47都道府県は `検証_スーパーコンビニ網羅性_都道府県別.csv`）

## 飲食店検証との対比（網羅性の全体像）

| 業態 | 最も網羅的なソース | 許可データの評価 |
|---|---|---|
| 飲食店 | 許可データ（管轄主体型の県）／POIも可 | 県により0.03〜4.36で不均質（`検証_飲食店網羅性_経済センサス突合.md`） |
| **コンビニ** | **Overture（≈実数98%）** | i2fas 49%＝不足 |
| **スーパー** | Overture/OSM（OSMがセンサスに一致） | 届出層で薄い |

→ **買い物難民診断（スーパー・コンビニ）ではPOI主・許可従が正しい**、を都道府県別データで確定。

## GMS（総合スーパー）の検証

農水省カテゴリ2「百貨店・総合スーパー(GMS)」。イオン・イトーヨーカドー等。

| ソース | 件数 | 備考 |
|---|---:|---|
| 経済センサス 561（百貨店・総合スーパー） | **1,023** | GMS＋百貨店（令和3は分離不可。百貨店約180→GMS約840） |
| Overture department_store | 2,336 | **ノイジー**（百貨店＋GMS＋Daiso等100均＋韓国百貨店の誤混入） |
| OSM department_store | 592 | 百貨店＋GMS |
| GMSチェーン名マッチ（Overture full） | 1,924 | うち**category=supermarket が1,074**＝GMSはスーパーに畳み込まれている。pharmacy408等ノイズ有 |

**結論**：**POIではGMSを独立カテゴリとして綺麗に分離できない**（department_storeは百貨店/100均/韓国混在、
イオン等の実店舗は大半がsupermarketカテゴリに入る）。→ **GMSは食料品店マスターでは実質「スーパー」に含まれて捕捉済み**。
別カテゴリとして厳密に数えたい場合は**センサス561を正**とし、POI側はブランド名キュレーションで識別するしかない。
買い物アクセス目的では、GMSの食料調達機能はスーパー捕捉で足りるため、無理な分離は不要。

## ドラッグストア（食料品扱い）の検証

農水省カテゴリ5。**調剤薬局(pharmacy)は除外、食料品を扱うドラッグストア(drugstore/chemist)が対象**
（Overtureの`drugstore` vs `pharmacy`区分、OSMの`shop=chemist` vs `pharmacy`区分が農水省の意図と一致）。

| ソース | 件数 | 対象/実数比 |
|---|---:|---|
| **業界実数①：商業動態統計 都道府県別店舗数(2021)** | **17,622** | 経産省・e-Stat `0004010388`（月次・チェーン全数、**都道府県別・再現可能**） |
| 業界実数②：JACDS（日本チェーンドラッグストア協会）実態調査 | 約21,000 | 全国値（会員ベースで広め） |
| 経済センサス 6031 ドラッグストア | 18,605 | 都道府県別（商業動態の89%〜JACDSの89%＝3ソース整合） |
| **Overture drugstore** | **7,731** | **44%（vs商業動態）／37%（vsJACDS）** |
| **OSM chemist** | **4,547** | **26%（vs商業動態）／22%（vsJACDS）** |
| （除外）Overture pharmacy＝調剤薬局 | 51,521 | 食料品扱わず対象外 |

**結論**：**POIはドラッグストアを大幅に取りこぼす**（Overture4割・OSM2〜3割）。コンビニ(Overture97%)と正反対で、
3つの実数ソース（商業動態17,622／センサス18,605／JACDS約21,000）どれで測っても同じ。
→ ドラッグストアは**商業動態/センサス6031/JACDSを主ソース**にし、POIは補完に留めるべき。
都道府県別カバー率（商業動態実数付き）は `検証_ドラッグストア網羅性_都道府県別.csv`。

## 農水省「食料品アクセス困難人口」5カテゴリの店舗マスター設計

本検証は、農水省の食料品アクセス推計で使う**食料品店マスターをどう作るか**の事前検証。5カテゴリと各ソースの対応：

| 農水省カテゴリ | センサス正解(令和3) | Overture | OSM | 実数の民間統計 |
|---|---|---|---|---|
| 1. 食品スーパー・各種食料品 | **581** 各種食料品小売業(22,378) | supermarket+grocery_store | supermarket 20,348 | **SM3団体 23,078店**（白書2024） |
| 2. 百貨店・総合スーパー(GMS) | **561**(1,023) | 無し⚠️ | department_store 592 | 日本百貨店協会/日本チェーンストア協会 |
| 3. コンビニ | 5891(22,714/過少) | convenience_store 54,987◎ | convenience 48,676 | **JFA 約56,000（正）** |
| 4. 生鮮専門店(青果/精肉/鮮魚) | **582+583+584** | butcher+seafood(青果無⚠️) | greengrocer+butcher+seafood | 単一団体なし→センサス主 |
| 5. ドラッグストア | **6031** ドラッグストア | drugstore 7,735 | chemist 4,549 | **JACDS 約21,000** |

- 単一ソースで5カテゴリを賄えない。**Overtureは青果・GMSを欠く**、生鮮はOSMがバランス良い。
- **マスターは「カテゴリごと最適ソースを選び Overture∪OSM を名寄せ統合」**が妥当。

## 正解データの信頼性（コンビニだけセンサス不可）

| カテゴリ | 正解に使うデータ | 理由 |
|---|---|---|
| コンビニ | **JFA約56,000** | センサス5891はFC店が本部分類に吸収され過少（実数の41%）。実数とのズレ機構が明確 |
| 食品スーパー | **センサス581**（業界実数23,078でクロスチェック済） | 直営チェーンが各店=1事業所計上でズレ機構なし。581=22,378が業界実数(SM3団体23,078)の97%で一致 |
| GMS・百貨店 | センサス561 | 同上。百貨店約180・GMSは日本チェーンストア協会でクロスチェック |
| 生鮮専門店 | センサス582/583/584 | 強い単一民間統計なし。センサスが主 |
| ドラッグストア | **JACDS約21,000**（センサス6031と併記） | 食品を扱うが分類の癖あり要確認。JACDS実態調査を実数基準に |

> 注：スーパーの「センサス=正」は**近似の一致**（センサス産業主分類581 vs 業界の売場面積・食品比率定義）で、
> 厳密証明ではない。SM年次統計調査の推計店舗数での最終裏取りが望ましい（本セッション未取得）。
> 民間統計出典：全国スーパーマーケット協会 年次統計調査/白書、日本チェーンドラッグストア協会(JACDS)、
> 日本フランチャイズチェーン協会(JFA)、日本百貨店協会、日本チェーンストア協会。

## 都道府県別「業界実数」の入手性（重要）

全国値だけでなく**都道府県別の業界実数**が要る場合の入手可否：

| カテゴリ | 都道府県別 業界実数 | 入手性 |
|---|---|---|
| **コンビニ** | **経産省 商業動態統計 都道府県別店舗数**（e-Stat `0004010382` 等、2015〜2024、月次・全数、2021=**56,352**） | ◎ **e-StatでAPI取得可・再現可能**（JFA約56,000と一致） |
| 食品スーパー | j-sosm「統計・データでみるスーパーマーケット」（SM3団体系・位置情報・都道府県別・月次） | △ サイトSSL証明書不良で自動取得に難。旧商業統計は2014で廃止 |
| 食品スーパー（代替） | **経済センサス581を都道府県別proxyに** | ◎ 全国で業界実数23,078の97%と実証済＝都道府県別も信頼できるproxy |
| ドラッグストア | JACDS実態調査（全国約21,000） | 全国値中心。都道府県別は要確認 |
| GMS・百貨店 | 日本百貨店協会（百貨店・都道府県別会員店）/日本チェーンストア協会 | 会員ベース |

→ **コンビニは商業動態の実数で都道府県別カバー率を出せる（実施済）。スーパーは自動取得の権威実数が乏しいため、
業界実数と97%一致するセンサス581を都道府県別proxyとして使うのが実務的。**

## 手法・注意（再現用）
- 空間結合: DuckDB spatial、`ST_Point(lon,lat)` を47都道府県ポリゴンに `ST_Contains`。取りこぼしは Overture 292/94,116・OSM 103/69,024（海岸線誤差、無視可）。
- Overture `operating_status` はこのdedupファイルでは全null＝**閉店店舗を除外できていない**（件数がやや上振れの可能性）。
- Overture `region` 列は表記不統一・非null38%で使えず、空間結合で都道府県を確定した。
- センサス: e-Stat `getStatsData` 表0004003257、tab=701-2021(事業所数)、cat01=5891/561/581、cat02=0(合計)。
- 次段（任意）: 市区町村境界で空間結合し、表0004005687の581等と市区町村別カバー率。
